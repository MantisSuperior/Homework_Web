HTML тэги

a:

Название тега: создание ссылок

Пример: Какая-то ссылка

Использование:

<a href="https://www.google.ru/"> Какая-то ссылка </a>

b:

Название тега: жирное начертание шрифта

Пример: Толстые буквы

Использование:

<b> Толстые буквы </b>

big:

Название тега: увеличение шрифта на единицу

Пример: Текст

Использование:

<big>Текст</big>

br:

Название тега: перевод строки

Пример:

ПАРУС

Белеет парус одинокой
В тумане моря голубом!..
Что ищет он в стране далекой?
Что кинул он в краю родном?...

Использование:

<p>ПАРУС</p>
<p>Белеет парус одинокой<br>
В тумане моря голубом!..<br>
Что ищет он в стране далекой?<br>
Что кинул он в краю родном?...</p>

center:

Название тега: выравнивание по центру

Пример:

Текст

Использование:

<center>Текст</center>

small:

Название тега: уменьшение текст на единицу

Пример: Текст

Использование:

<small>Текст</small>

strong:

Название тега: акцентирования текста (в браузере отображается текст жирным начертанием)

Пример: Текст

Использование:

<strong>Текст</strong>

sup:

Название тега: отображение шрифта в виде верхнего индекса (шрифт отображается выше базовой линии текста и уменьшенного размера)

Пример: Текст

Использование:

<sup>Текст</sup>

sub:

Название тега: отображение шрифта в виде нижнего индекса (шрифт отображается ниже базовой линии текста и уменьшенного размера)

Пример: Текст

Использование:

<sub>Текст</sub>

table:

Название тега: создание таблиц

Пример:

Табличное значение

Использование:

<table border = "1">
<tr>
<td>Табличное значение</td>
</tr>
</table>

td:

Название тега: создание одной ячейки таблицы (тег td должен размещаться внутри контейнера tr, который в свою очередь располагается внутри тега table)

Пример:

Табличное значениеТабличное значениеТабличное значение

Использование:

<table border = "1">
<tr>
<td>Табличное значение</td><td>Табличное значение</td><td>Табличное значение</td>
</tr>
</table>

tr:

Название тега: создание строки таблицы (каждая ячейка в пределах такой строки может задаваться с помощью тега td)

Пример:

Табличное значение
Табличное значение
Табличное значение

Использование:

<table border = "1">
<tr>
<td>Табличное значение</td>
</tr>
<tr>
<td>Табличное значение</td>
</tr>
<tr>
<td>Табличное значение</td>
</tr>
</table>

ul:

Название тега: маркированный список (каждый элемент списка должен начинаться с тега li)

Пример:

  • элемент маркированного списка
  • элемент маркированного списка
  • элемент маркированного списка

Использование:

<ul>
<li>элемент маркированного списка</li>
<li>элемент маркированного списка</li>
<li>элемент маркированного списка</li>
</ul>

ol:

Название тега: нумерованный список (каждый элемент списка должен начинаться с тега li)

Пример:

  1. элемент маркированного списка
  2. элемент маркированного списка
  3. элемент маркированного списка

Использование:

<ol>
<li>элемент маркированного списка</li>
<li>элемент маркированного списка</li>
<li>элемент маркированного списка</li>
</ol>

li:

Название тега: определение отдельного элемента списка (внешний тег ul или ol устанавливает тип списка — маркированный или нумерованный)

Пример:

  • элемент маркированного списка
  1. элемент нумерованного списка

Использование:

<ul>
<li>элемент маркированного списка</li>
</ul>
<ol>
<li>элемент нумерованного списка</li>
</ol>

font:

Название тега: контейнер для изменения характеристик шрифта, таких как размер, цвет и гарнитура

Пример: Текст

Использование:

<font size="15" color="green" face="Georgia">Текст</font>

h1:

Название тега: заголовок первого уровня

Пример:

Заголовок первого уровня

Использование:

<h1>Заголовок первого уровня</h1>

Пример:

Заголовок второго уровня

<h2>Заголовок второго уровня</h2>

Заголовок третьего уровня

Использование:

<h3>Заголовок третьего уровня</h3>

h4:

Название тега: заголовок четвёртого уровня

Пример:

Заголовок четвёртого уровня

Использование:

<h4>Заголовок четвёртого уровня</h4>

h5:

Название тега: заголовок пятого уровня

Заголовок пятого уровня

Использование:

h6:

Пример:

Заголовок шестого уровня

img:

Название тега: отображение на веб-странице изображений в графическом формате GIF, JPEG или PNG (адрес файла с картинкой задаётся через атрибут src)

Пример:

Использование:

<img src="https://vignette.wikia.nocookie.net/lotr/images/f/fb/3ifu.jpg/revision/latest?cb=20130423140938&path-prefix=ru" width="300" height="300">

iframe:

Название тега: создание плавающего фрейма

Пример:

Использование:

<iframe src="https://www.apple.com/" width="468" height="60" align="left"></iframe>

u:

Название тега: подчёркивание текста

Пример: Текст

Использование:

<u>Текст</u>

i:

Название тега: курсивное начертание шрифта

Пример: Текст

Использование:

<i>Текст</i>

title:

Название тега: заголовок документа (title не является частью документа и не показывается напрямую на веб-странице)

Пример: Заголовок

Использование:

<title>Заголовок</title>
